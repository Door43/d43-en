===== Translation Notes: =====

This begins the list of the gatekeepers.

  * **Korahites** - See how you translated this in [[:en:bible:notes:1ch:09:17|1 Chronicles 9:19]].
  * **Meshelemiah** - See how you translated this in [[:en:bible:notes:1ch:09:20|1 Chronicles 9:21]].
  * **Kore** - See how you translated this in [[:en:bible:notes:1ch:09:17|1 Chronicles 9:19]].
  * **Asaph** - This is an alternative spelling to "Ebiasaph" in [[:en:bible:notes:1ch:09:17|1 Chronicles 9:19]].
  * **Zechariah** - See how you translated this in [[:en:bible:notes:1ch:09:20|1 Chronicles 9:21]].
  * **Jediael...Zebadiah...Jathniel...Elam...Jehohanan...Eliehoenai** - These are the names of men.  (See: [[en:ta:vol1:translate:translate_names]])
  * **the second … third … fourth … seventh** - This shows the order in which the sons were born. AT: "number 2...number 3...number 4...number 7" or "the next" for each one.  (See: [[en:ta:vol2:translate:translate_numbers]])
