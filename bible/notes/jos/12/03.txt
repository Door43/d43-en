===== translationNotes: =====

  * **Sihon** - This was the name of a defeated Amorite king. Translate the same way as [[:en:bible:notes:jos:02:10|Joshua 2:10]] (See: [[:en:ta:vol1:translate:translate_names]])
  * **Og** - This was the name of a defeated king of Bashan. Translate the same way as in [[:en:bible:notes:jos:02:10|Joshua 2:10]].
  * **the Rephaim** - This is the name of a people group. 
  * **the Arabah** - This is the name of a place. Translate the same way as in [[:en:bible:notes:jos:03:14|Joshua 3:16]].
  * **Sea of Chinnereth** - This is a place. Translate the same way as in [[:en:bible:notes:jos:11:01|Joshua 11:2]].
  * **Beth Jeshimoth ... Mount Pisgah ... Ashtaroth ... Edrei ... Salecah** - These are the names of places.
  * **Ashtaroth** - This is the name of a place. Translate the same way as in [[:en:bible:notes:jos:09:09|Joshua 9:10]].
  * **Maacathites** - This is the name of a group of people.
  * **Heshbon** - This is the name of a place. Translate the same way as in [[:en:bible:notes:jos:12:01|Joshua 12:2]].
