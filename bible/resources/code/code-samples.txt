====== Sample Door43 Code ======

Code to pull in content from a specific section:

<code>
{{section>:en:bible:notes:luk:01:01#important terms:&noheader}}
</code>

Code to display the activity of a user:

  * Replace "USER" with Door43 username
  * Count= sets number of results

<code>
**USER:**
{{changes>user=USER&count=12&render=pagelist(header,date,user,nocomments)&maxage=25920000}}
</code>

Display pages with in namespace with specific tag:

<code>
{{topic>en:bible:notes:luk?draft&nocomments&simplelist}}
</code>

Search ns for pages with or without text string:

<code>
{{pagequery>ns:en:bible:notes:mat "{{tag>create" -"{{tag>delete"; fulltext; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}
</code>

Search for inserted beginning **or**  ending <nowiki>HTML <font></nowiki> code in OBE pages:
<code>
{{pagequery>ns:en:obe "<font" | ns:en:obe "</font>"; fulltext; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}
</code>

Search for inserted beginning **or**  ending <nowiki>HTML <font></nowiki> code in Bible notes pages:
<code>
{{pagequery>ns:en:bible:notes "<font" | ns:en:bible:notes "</font>"; fulltext; sort=title:asc; display={title}; natsort; layout=table; showcount; hidemsg; hidejump;}}
</code>

Open Door43 page in edit mode:

<code>
?do=edit&rev=0
</code>
