===== Metadata Encoding Software =====

//This page answers the question:// //What software do you recommend for encoding metadata?//

The following software is recommended for coding the metadata properties of MP3 and MP4 files.

==== Windows OS ====

  * **[[http://www.mp3tag.de/en/download.html|Mp3Tag]]** - For writing and editing file tags/meta data. Mp3Tag is our recommendation for a free program that works for mp3 and mp4 files.
