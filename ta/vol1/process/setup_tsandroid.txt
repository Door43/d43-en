===== Setting Up translationStudio-Android =====

This module answers the question: How do I set up translationStudio-Android?\\
Before you start this module have you learned about:
  * [[en:ta:vol1:process:platforms|Selecting a Platform]]

==== Set up the app ====

  - First you will need to install the app on an Android device. For instructions see [[en:ta:vol1:tech:ts_install]]. (See also https://unfoldingword.org/translationstudio/ )
  - First time use of translationStudio will require an agreement to the [[en:ta:vol1:intro:statement_of_faith]], the [[en:ta:vol1:intro:translation_guidelines]], and the [[en:ta:vol1:intro:open_license]]. For help see [[en:ta:vol1:tech:ts_first]].

==== Set up the source language ====

  - [[en:ta:vol1:tech:ts_select]]. Go to [[en:ta:vol1:tech:ts_request]] if you cannot find your language.
