===== Selecting a Platform =====

Before you start this module have you learned about: [[en:ta:vol1:process:getting_started|Getting Started]]

Now you need to select a translation and checking platform. This is where the translation and checking teams will do their work. There are four options: 
    
   * Online in a web browser - For those with consistent internet, they can work straight on door43. Click [[en:ta:vol1:process:setup_door43|here]] to set up door43 for translation.
   * Offline on a mobile device - For those with access to a tablet or mobile device, they can work in our translation app called translationStudio. Click [[en:ta:vol1:process:setup_tsandroid|here]] to set up translationStudio.
   * Offline on a computer - For those without consistent internet but with access to a computer, they can work in translationStudio-Desktop. Click [[en:ta:vol1:process:setup_tsdesktop|here]] to set up translationStudio-Desktop.
   * Offline in a word processor - For those who would like to work in a word processor, they download the files needed to do that. Click [[en:ta:vol1:process:setup_word|here]] to set up word processor.
